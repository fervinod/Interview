NOTE:

ssacli fully works with Gen10 servers, but hpasmcli does not.Starting with Gen10 architecture,
HPE changed how system health information is communicated, directly affecting these utilities.

ssacli (Smart Storage Administrator CLI): Manages storage controllers, hard drives (HDDs/SSDs), RAID arrays, and logical volumes.

#ssacli monitors: Physical disks, RAID controller cache, battery backup units (Smart Storage Battery), logic drives, and encryption settings.

hpasmcli (HP Management CLI): Manages physical server chassis components, environmental status, and system architecture logs.
#hpasmcli monitors: Cooling fans, temperature sensors, power supplies, CPU/DIMM slots, and the Integrated Management Log (IML).

#instead of hpasmcli on Gen10 Use ilorest (The modern CLI replacement):


ILOREST:
The HPE RESTful Interface Tool (ilorest) utilizes the Redfish API to interact directly with the iLO controller. It can be run locally inside the host operating system using the chifublob driver wrapper, or executed remotely over a network by defining login flags (--url, -u, -p)


ilorest serverinfo --type=Chassis: Outputs structural details on real-time server conditions, fan metrics, power redundancy status, and thermal arrays.

ilorest serverinfo --type=System: Displays high-level specifications including current memory capacity, CPU core types, and model serial identifiers.

ilorest systemlogs --selectlog=IML: Pulls the raw content logs from the integrated Integrated Management Log (IML) for hardware fault tracking
